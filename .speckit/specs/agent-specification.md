# Agent Specification

## Coordinator Agent

Purpose:
Manage all agents and combine results.

Input:
Business Idea

Output:
Aggregated Analysis

---

## Market Agent

Purpose:
Analyze market potential.

Output:
Market Score (0-100)

---

## Feasibility Agent

Purpose:
Analyze technical feasibility.

Output:
Feasibility Score (0-100)

---

## Risk Agent

Purpose:
Identify risks.

Output:
Risk Score (0-100)

---

## Scoring Agent

Purpose:
Generate final validation score.

Output:
Overall Score

---

## Report Agent

Purpose:
Generate final report.

Output:
Validation Report
