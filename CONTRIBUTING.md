# Contribution Guidelines

## Branch Strategy

main
develop
feature/*
bugfix/*

## How to Contribute

1. Fork Repository
2. Create Feature Branch
3. Commit Changes
4. Push Branch
5. Create Pull Request

## Coding Standards

### Python
- PEP8
- Type Hints
- Docstrings

### Frontend
- ESLint
- Prettier

## Testing Requirements

- Unit Tests Mandatory
- Agent Tests Required
- Minimum 80% Coverage

## Pull Request Checklist

- Tests Passed
- Documentation Updated
- Specification Updated
